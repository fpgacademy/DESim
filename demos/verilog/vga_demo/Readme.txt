In this demo:

-- Uses the VGA Display

To use:

1. Toggle Key[0] to start the VGA display
2. The VGA display will cycle through various colors

The VGA resolution can be set to one of 640x480, 320x240, or 160x120 in the
file resolution.v. Ensure that the same setting is made in the VGA Display
pane in the DESim GUI.
