This demo contains the complete set of DESim's inputs and outputs.
It can be used a template for further projects.

In this demo:

-- The LEDR display a 010101010101 pattern
-- The word HELLO is written on the HEX displays
-- The VGA output shows an object that is read from a memory

To use: Toggle KEY[0]

Note: the VGA Display resolution in the DESim GUI must match the resolution
select in the file resolution.v.
