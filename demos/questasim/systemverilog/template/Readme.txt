This demo contains the complete set of DESim's inputs and outputs.
It can be used a template for further projects.

In this demo:

-- The LEDR display a 010101010101 pattern
-- The word HELLO is written on the HEX displays
-- SW[3:0] is used for the VGA's X coordinate
-- SW[7:4] is used for the VGA's Y coordinate
-- KEY[3:1] is used for the VGA's color
-- KEY[0] is used for the VGA's plot control

To use:

1. Set SW[7:0] and KEY[3:1] to any value
2. Toggle KEY[0]
3. Repeat the above steps
